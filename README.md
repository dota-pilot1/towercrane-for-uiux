# towercrane-for-uiux

UI/UX 시스템과 내부 관리자형 프론트엔드 패턴을 정리하기 위한 저장소입니다.

## Projects

- `towercrane-for-uiux-front`
  - Vite + React + TypeScript
  - Tailwind CSS + Radix UI
  - Zustand + react-hook-form + zod
  - React Query + TanStack Table

## Getting Started

```bash
cd towercrane-for-uiux-front
pnpm install
pnpm dev
```
