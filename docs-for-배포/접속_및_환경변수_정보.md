## 프론트 환경변수

파일 위치: `towercrane-for-uiux-front/.env.production`


Vite 환경변수는 빌드 시점에 번들에 포함됩니다. 값을 바꾸면 반드시 프론트를 다시 빌드하고 배포해야 합니다.

## 보안 원칙

- AWS Secret, OpenAI/Gemini Key, 서버 `.env` 값은 문서에 직접 쓰지 않습니다.
- SSH Key는 권한 `400`으로 유지합니다.
- 서버 `.env`는 EC2 안에서 직접 확인/수정합니다.
- 프론트 정적 배포용 AWS credential과 백엔드 파일 업로드용 AWS credential은 용도가 다를 수 있으니 혼용 여부를 배포 전 확인합니다.
